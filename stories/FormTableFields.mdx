# Fields For FormTable

These components are field areas that are inside of a `<tr>` element.

## Using Fields Inside FormTable

With `MDX` we can define a story for `Checkbox` right in the middle of our
markdown documentation.

<Canvas>
  <Story name="Using With FormTable Component">
    <Form id={'form-id'} onSubmit={onSubmit}>
      <FormTable >
          <>
              <!-- Fields -->
               <TrSubmitButton
                  id={'submit-button'}
                  name={'submit-button'}
                  value={'Save'}
              />
          </>
      </FormTable>
    </Form>
  </Story>
</Canvas>
